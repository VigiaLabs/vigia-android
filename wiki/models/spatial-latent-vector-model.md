# SpatialLatentVector

```kotlin
data class SpatialLatentVector(
    val dimensions: Int,           // 256 (REQUEST_256D) or 512 (REQUEST_512D)
    val data: FloatArray,
    val originTimestampMs: Long,
)
```

Generated by the Pi 5 road-hazard CV model. Transmitted over BLE GATT TELEMETRY\_CHAR. Included verbatim in `VigiaSearchContext` so VIGIASearch backend can perform spatial similarity lookups.
